# 简介
> 一款基于Flutter框架开发的跨平台实时通讯社交软件

> 默讯社交软件，实时通讯，想你所想，做你所能！

# 适配平台
- Android 9及以上
- Ios 11及以上
  
> 未来计划添加支持MacOs Windows端

# 功能列表
- 群聊&好友消息
- 个人主页
- 聊天通讯
- 添加好友
- 创建群聊
- 修改头像
